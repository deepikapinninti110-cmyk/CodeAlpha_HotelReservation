🏨 Hotel Reservation System (Java)
A Java-based Hotel Reservation System developed using Object-Oriented Programming (OOP) concepts.
This project is created as part of the CodeAlpha Java Programming Internship (Task 4).
📌 Project Overview
The Hotel Reservation System allows users to:
View available hotel rooms
Book rooms by room number
Cancel existing reservations
View current reservations
The application runs in a console-based environment and simulates real-world hotel booking functionality.
✨ Features
View available rooms
Room categorization:
Standard
Deluxe
Suite
Book hotel rooms
Cancel reservations
View all current reservations
Price display (payment simulation)
Menu-driven console interface
🛠️ Technologies Used
Java
VS Code
JDK 8 or higher
